# GUI_CALCULATOR_CPP
This is a desktop-based calculator application featuring a Graphical User Interface. It is designed to provide a simple and intuitive way for users to perform calculations without using a command-line interface.
Key Features
Basic Arithmetic: Supports addition, subtraction, multiplication, and division.
User-Friendly Interface: Responsive buttons and a clear display for easy navigation.
Error Handling: Prevents crashes from invalid inputs (like dividing by zero).
Clear Function: Quickly reset the calculator with a single "C" or "AC" button.
